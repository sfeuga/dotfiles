# My Dotfiles

1. Install stow (`brew install stow`)
2. Clone this repository (`git clone git@github.com:sfeuga/dotfiles.git ~/.dotfiles`)
3. Run `stow -d ~/.dotfiles asdf curl fish fonts git kitty ledger less nvim ruby rvm ssh zsh`
